# Sales Dashboard Redesign

## Project Overview
This project was completed as part of a data visualization assignment using Power BI.

The original dashboard and dataset were provided as part of the coursework. The objective of this project was to redesign and improve the dashboard by making it cleaner, more organized, and easier to understand.

The focus of this project was improving the presentation of information so users can quickly identify important insights and understand the data more efficiently.

---

## Dataset Information

The dataset contains sales-related information with 80 records and 26 attributes.

Main data categories include:

- Product details
- Order information
- Customer information
- Sales performance
- Shipping information
- Customer feedback

Example fields:

- Product Name
- Category
- Sales Amount
- Quantity
- Customer Name
- Order Date
- Shipping Status
- Estimated Delivery Date
- Feedback Score
